Uso y manejo Flex Box, CSS Grid

Esta página pretende dar una guía completa explicar todo sobre flexbox, centrándose en todas las diferentes propiedades posibles para el elemento principal (el contenedor flexible) y los elementos secundarios (los elementos flexibles). También incluye historial, demostraciones, patrones y una tabla de soporte del navegador.

La comunicada de desarrollo de Developer Mozilla (2019) nos indica que:
"La diferencia básica entre CSS Grid Layout y CSS Flexbox Layout es que flexbox fue diseñado para el diseño en una dimensión, ya sea una fila o una columna. La cuadrícula se diseñó para un diseño bidimensional: filas y columnas al mismo tiempo. Sin embargo, las dos especificaciones comparten algunas características comunes, y si ya ha aprendido a usar flexbox, las similitudes deberían ayudarlo a familiarizarse con Grid." (p. 87).

Diseño unidimensional versus bidimensional segun Developer Mozilla (2019):
"Un ejemplo simple puede demostrar la diferencia entre diseños unidimensionales y bidimensionales. En este primer ejemplo, estoy usando flexbox para diseñar un conjunto de cajas. Tengo cinco elementos secundarios en mi contenedor, y he dado los valores de las propiedades flexibles para que puedan crecer y encogerse desde una base flexible de 150 píxeles. También configuré la flex-wrappropiedad en wrap, de modo que si el espacio en el contenedor se vuelve demasiado estrecho para mantener la base flexible, los elementos se ajustarán a una nueva fila."
